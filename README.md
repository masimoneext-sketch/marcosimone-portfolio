# 🌐 Portfolio Personale — Marco Simone

Portfolio professionale personale. Presenta competenze, progetti, esperienze e soluzioni sviluppate.

**Live:** [marcosimone-life.marcosimone.tech](https://marcosimone-life.marcosimone.tech)

## Stack tecnico

| Tecnologia | Dettaglio |
|-----------|----------|
| React 18 | UI library |
| Vite | Build tool |
| TypeScript | Type safety |
| Tailwind CSS | Styling |

## Sezioni

- **Hero** — presentazione e call to action
- **About** — profilo professionale
- **Skills** — tecnologie e competenze
- **Projects** — progetti realizzati con mockup interattivi
- **Experience** — esperienze lavorative
- **Solutions** — soluzioni e portali sviluppati
- **Contact** — form di contatto

## Componenti

`Hero`, `About`, `Skills`, `Projects`, `Experience`, `Solutions`, `Contact`, `Navbar`, `Footer`, `Splash`, `GlobalParticles`, `ProjectMockup`

## SEO

- JSON-LD strutturato
- `sitemap.xml`
- `robots.txt`
- Google Search Console verificato

## Sviluppo locale

```bash
npm install
npm run dev
```

## Build e deploy

```bash
npm run build
```

La cartella `dist/` viene servita tramite il web server configurato sul VPS.
